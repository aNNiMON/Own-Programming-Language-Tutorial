# Циклы

## Цикл while

```own
while условие {
   тело цикла
}
```

Скобки в условии необязательны.

```own
i = 0
while i < 5 {
  print i++
}

// или

i = 0
while (i < 5) {
  print i++
}
```

## Цикл do-while

```own
do {
   тело цикла
} while условие
```

Скобки в условии необязательны.

```own
i = 0
do {
  print i++
} while i < 5

// или

i = 0
do {
  print i++
} while (i < 5)
```

## Цикл for

```own
for инициализация, условие_работы, инкремент {
   тело цикла
}

for (инициализация, условие_работы, инкремент) {
   тело цикла
}
```

Скобки в условии необязательны.

```own
for i = 0, i < 5, i++
  print i++

// или

for (i = 0, i < 5, i++) {
  print i++
}
```

## Цикл foreach

Перебирает элементы строки, массива или карты.

Перебор строки:

```own
for символ : строка {
   тело цикла
}
for символ, код : строка {
   тело цикла
}
```

Перебор массива:

```own
for значение : массив {
   тело цикла
}
for значение, индекс : массив {
   тело цикла
}
for (значение : массив) {
   тело цикла
}
```

Перебор карты:

```own
for (ключ, значение : карта) {
   тело цикла
}
```

Скобки необязательны.
